# JavaFirstCode

This repository contains a simple Java program demonstrating the classic **"Hello, World!"** example.

## 📁 Project Structure

- `HelloWorld.java` – Basic Java program that prints "Hello, world!" to the console.
- `src/first/` – Java source folder.

## 💡 Purpose

This project is intended as a Git test case and entry point for pushing Eclipse Java code to GitHub using Git CLI and Personal Access Tokens.

## 🛠️ Built With

- Java 17 (or your installed version)
- Eclipse IDE
- Git CLI

## 🚀 How to Run

1. Open the project in Eclipse or any Java IDE.
2. Navigate to `HelloWorld.java`.
3. Run the program.

## 🔐 GitHub Setup Notes

This project was pushed using a **Classic GitHub Personal Access Token (PAT)** via HTTPS authentication.
